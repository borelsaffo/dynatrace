# dynatrace

# Plan de préparation – Certification Dynatrace Associate

## Objectif
Préparer efficacement la certification Dynatrace Associate en 2 à 4 semaines selon ton niveau.

---

## Prérequis recommandés
- Bases en monitoring / observabilité
- Compréhension de :
  - Kubernetes / Containers
  - Applications Web / APIs
  - Logs / Metrics / Traces
  - Cloud / Infra basics

---

## Semaine 1 — Fondamentaux Dynatrace
### À apprendre
- Architecture Dynatrace
- OneAgent
- ActiveGate
- Dynatrace SaaS vs Managed
- Smartscape & Topology

### Ressources
- Dynatrace University – Associate Learning Path
- Documentation officielle Dynatrace

### Objectif
Être capable d’expliquer comment Dynatrace collecte et structure les données.

---

## Semaine 2 — Observabilité & Analyse
### À apprendre
- Distributed Tracing
- Service Flow / PurePath
- Davis AI
- Root Cause Analysis
- Metrics / Dashboards / Alerts

### Pratique
- Naviguer dans un tenant Dynatrace démo/lab
- Créer dashboards simples
- Analyser incidents simulés

---

## Semaine 3 — Monitoring Infrastructure & Kubernetes
### À apprendre
- Monitoring Hosts / Processes
- Kubernetes Monitoring
- Namespace / Workloads / Pods
- Tags / Management Zones

### Pratique
- Déployer OneAgent sur cluster K8s
- Vérifier remontée des métriques

---

## Semaine 4 — Révision & Exam Prep
### À faire
- Relecture des notes
- Examens blancs / quiz
- Revoir les points faibles

### Focus
- Terminologie officielle Dynatrace
- Différences entre composants
- Cas d’usage typiques

---

## Routine Hebdomadaire Conseillée
- **Lundi–Vendredi :** 1h / jour théorie
- **Week-end :** 2–3h de pratique / labs

---

## Conseils pour l’examen
- Bien connaître le vocabulaire Dynatrace exact
- Lire attentivement les questions “best answer”
- Maîtriser les cas d’usage plutôt que le par cœur

---

## Checklist Finale
- [ ] Architecture comprise
- [ ] OneAgent / ActiveGate maîtrisés
- [ ] Smartscape compris
- [ ] Davis AI compris
- [ ] Kubernetes monitoring compris
- [ ] Dashboards / Alerts pratiqués
- [ ] Quiz blancs > 80%

---

## Durée Estimée Totale
**20 à 35 heures** de préparation pour quelqu’un avec background DevOps/SRE.
